Synthetic Custdev

Понятный интерфейс для синтетического кастдева без лишнего шума.

Поток простой: добавьте контекст и запустите исследование. Supabase и GitHub подключаются здесь же, внутри этого локального проекта.

1. Дайте контекст Опишите продукт, аудиторию, гипотезы или загрузите документы.
2. Запустите исследование Система покажет статусы и последовательно прогонит этапы прямо в этом интерфейсе.

Настройки

Минимум полей, только проект и сценарий.

Чат исследования

Пишите вводные сообщением. При необходимости добавляйте документы. После этого запускайте исследование.

Документы можно добавлять прямо в чат. После загрузки они сразу участвуют в исследовании.

Результат

Краткий отчет для чтения. Полный пакет для хранения всех сообщений, документов и сырого контекста.
